# 🧠 Memory Game

A browser-based memory matching game built with HTML, CSS, and Vanilla JavaScript. The objective is to find all matching pairs by remembering the positions of hidden cards while using the fewest possible moves.

---

## ✨ Features

* 4×4 card grid
* Random card shuffle
* Move counter
* Match tracking
* Restart game
* Responsive layout

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)

---

## ⚙️ How It Works

The game generates eight pairs of matching cards and shuffles them using the Fisher–Yates algorithm. Players reveal two cards at a time to find matching pairs. Matched cards remain visible, while unmatched cards flip back over after a short delay. The game ends when all pairs have been successfully matched.
