# 🧠 Memory Game

A browser-based memory matching game built with HTML, CSS, and Vanilla JavaScript. The objective is to find all matching pairs while using the fewest possible moves.

---

## ✨ Features

- 4×4 card grid
- Random card shuffle
- Move counter
- Match counter
- Restart game
- Responsive layout

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)

---

## ⚙️ How It Works

1. The game generates eight pairs of cards.
2. The deck is shuffled randomly.
3. Players reveal two cards each turn.
4. Matching pairs remain visible.
5. Non-matching cards flip back.
6. The game ends when all pairs are matched.
