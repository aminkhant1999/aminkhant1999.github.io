# ❌⭕ Tic-Tac-Toe

A classic two-player Tic-Tac-Toe game developed using HTML, CSS, and Vanilla JavaScript.

---

## ✨ Features

- Two-player gameplay
- Win detection
- Draw detection
- Restart game
- Dynamic player indicator
- Responsive design

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)

---

## ⚙️ How It Works

Players take turns placing X and O on a 3×3 board. After every move, the game checks all winning combinations. If no winning condition is met and the board is full, the game ends in a draw.
