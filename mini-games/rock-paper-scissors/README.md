# 🪨📄✂️ Rock Paper Scissors

A browser-based Rock Paper Scissors game where players compete against a computer opponent using randomly generated moves.

---

## ✨ Features

- Computer opponent
- Random move generation
- Best-of match selection
- Live scoreboard
- Reset game
- Responsive UI

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)

---

## ⚙️ How It Works

The player selects Rock, Paper, or Scissors. The computer randomly generates its move. The game compares both selections, updates the score, and continues until one side reaches the selected target score.
